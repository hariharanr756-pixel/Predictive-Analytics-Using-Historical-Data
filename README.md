# 📊 Predictive Analytics Using Historical Data

## 📌 Project Overview

This project builds a predictive model using historical sales data to forecast future trends.

## 🎯 Objective

To analyze past data and predict future sales using machine learning techniques.

## 🛠️ Tools & Technologies

* Python
* Pandas
* Matplotlib
* Scikit-learn
* Jupyter Notebook

## 📂 Dataset

The dataset contains:

* Date
* Sales

## ⚙️ Steps Performed

1. Data loading using Pandas
2. Data cleaning and preprocessing
3. Data visualization (trend analysis)
4. Model building using Linear Regression
5. Model evaluation using Mean Squared Error
6. Future prediction of sales

## 📈 Output

* Sales trend graph
* Actual vs Predicted comparison
* Future sales predictions

## 📊 Result

The model successfully predicts future sales based on historical trends.

## 🚀 How to Run

1. Open Jupyter Notebook
2. Run all cells in `Predictive_Analytics_Project.ipynb`
3. View outputs and graphs

## 📌 Conclusion

Predictive analytics helps in understanding trends and making data-driven decisions.
